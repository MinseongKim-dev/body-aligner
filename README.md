# BodyAlign — 체형 분석 기반 맞춤 운동 추천 플랫폼

체크리스트 + 사진(MediaPipe Pose) 분석으로 체형 문제를 파악하고, 유튜브 자막에서 운동을 자동 추출하여 맞춤 교정 루틴을 제공합니다.

## 주요 기능

- **자가 체크리스트** — 15가지 항목으로 체형 증상 파악
- **바디맵** — 통증·불편 부위 클릭 선택
- **사진 포즈 분석** — MediaPipe Pose로 어깨·골반·무릎·발 자동 측정 (브라우저만, 서버 전송 없음)
- **패턴 감지** — 발목→무릎→골반 연쇄 패턴 우선순위 도출
- **유튜브 자막 추출** — 자막 키워드 자동 매칭 (LLM 비용 없음)
- **맞춤 루틴** — Phase 1→2→3 단계별 교정 루틴

## 비용: 완전 무료

LLM API 없음 · MediaPipe 브라우저 실행 · Vercel 무료 플랜 가능

## 실행

```bash
npm install && npm run dev
```

> ⚠️ 이 서비스는 의료 진단을 대체하지 않습니다.
